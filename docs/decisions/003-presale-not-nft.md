# ADR-003：产品化预售，而非标准化 NFT 发行

> 状态：已采纳 · 2026-07-01

---

## 背景

早期讨论中，第三阶段曾设想为「NFT / 链上凭证发行」。用户明确：**链上/NFT 为后期**，当前聚焦售前；发行形态应改为 **定制化 · 产品化预售**。

---

## 决策

第三阶段能力定义为：

- **定制化**：每位艺术家/系列单独设计预售 SKU
- **产品化**：明确交付物、名额、权益边界，类似可售卖的文献包/支持包
- **非标准化发行**：不做统一 token 模板、不做见作品就铸凭证

链上/NFT 若未来引入，仅作为编号凭证或关系锚点，**不是**验证期前提。

---

## 后果

- 产品需预留「预售 SKU / 预售页」模型，但验证期可人工配置
- 文档、对外表述统一使用「产品化预售」，避免 NFT 平台误解
- 支付与分账推迟到预售试点验证之后
