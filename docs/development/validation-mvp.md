# 验证期 MVP 范围

> 4–6 周 · 售前阶段 · 不含链上/NFT/完整支付

---

## 1. 目标

验证：**艺术家资产档案**是否能让第三方更容易理解艺术家，并自然长出支持关系。

---

## 2. 功能范围

### ✅ 必做

| 功能 | 说明 |
|------|------|
| 艺术家资产档案页 | 6 模块（见 mvp-modules.md） |
| 作品结构化展示 | 硬资产字段、系列归属 |
| 系列与时间线 | 核心差异化 |
| 过程资产展示 | 至少公开层 |
| 平台评级/策展短评 | 轻量，辅助理解 |
| 关注/收藏 | 支持者关系起点 |
| 合作/咨询意向 | 可导向定制预售咨询 |
| AI 辅助 | 标签、简介、叙事草稿（可选） |
| 响应式 Web | PC + 手机 |

### 🔜 验证后期（人工为主）

| 功能 | 说明 |
|------|------|
| 定制化产品化预售 | 1 位艺术家 × 1 种 SKU 试点 |
| 预售页模板 | 人工配置，可不接支付 |

### ❌ 不做

- NFT / 区块链 / 链上存证
- 完整支付与分账
- 艺术估值/投资模型
- 标准化 token 发行
- 工厂授权交易（旧方向，后期可选）

---

## 3. 技术范围

- Next.js 15 + TypeScript + Tailwind
- 本地 MySQL + Prisma
- 本地图片存储
- Mock 认证或手机号登录（MOCK_AUTH）
